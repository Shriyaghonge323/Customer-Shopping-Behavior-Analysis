# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes transactional data from 3,900 customers to uncover insights into spending patterns, demographics, and subscription behavior.

## Tech Stack
* **SQL Server:** Data cleaning and exploratory analysis.
* **Power BI:** Interactive dashboarding and visualization.
* **Python (Pandas):** Initial data preparation and missing value imputation.

## Key Insights
* **Gender Gap:** Male customers generate 2.1× more revenue than female customers.
* **Top Performers:** Clothing is the leading category, with Young Adults ($18-35$) contributing the highest revenue.
* **Loyalty:** 80% of the customer base are "Loyal" buyers, while "New" customers represent a key growth opportunity.
